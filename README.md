# Playbacc

A self-hostable music tracking and analytics platform for music enthusiasts who want to own their listening data.

## Packages

- **API**: `packages/api`
- **Web**: `packages/web`
- **Shared**: `packages/shared`

## Development

```bash
bun install
bun run dev
```

Or individually:

```bash
bun run dev:api
bun run dev:web
```
